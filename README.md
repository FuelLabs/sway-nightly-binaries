# Fuel Nightly Binaries

Every day at **00:00 UTC**, the workflows in this repository check out
the [sway], [fuel-core], and [forc] repositories, build the binaries,
and create nightly releases.

These binaries are made available through [fuelup].

## Source Repositories

| Binary | Repository |
|--------|------------|
| `forc` and related binaries | [sway] |
| `fuel-core` | [fuel-core] |
| `forc-wallet`, `forc-crypto`, `forc-node`, `forc-client` | [forc] |

[sway]: https://github.com/FuelLabs/sway
[fuel-core]: https://github.com/FuelLabs/fuel-core
[forc]: https://github.com/FuelLabs/forc
[fuelup]: https://github.com/FuelLabs/fuelup
